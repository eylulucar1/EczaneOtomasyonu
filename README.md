# EczaneOtomasyonu

Basit bir Eczane Otomasyonu Windows Forms uygulaması (.NET Framework 4.7.2, C# 7.3).

Özellikler:
- Hasta kayıt yönetimi
- İlaç ekleme, listeleme, silme
- Güvence (sigorta) ekleme

Kurulum:
1. Visual Studio 2019 veya daha yeni bir sürüm (ör. Visual Studio Community 2026) yükleyin.
2. Depoyu klonlayın:
   `git clone https://github.com/eylulucar1/EczaneOtomasyonu.git`
3. Proje dizininde `EczaneDb.accdb` veritabanı dosyasını proje köküne kopyalayın veya `app.config` içindeki bağlantı dizesini güncelleyin.
4. Visual Studio ile `EczaneOtomasyonu.sln` dosyasını açın ve projeyi derleyin.

Dikkat:
- `EczaneDb.accdb` veritabanı dosyası güvenlik nedeniyle .gitignore ile hariç tutulmuştur; repoya göndermeyin.
- Bağlantı dizelerini kaynak kontrolüne koymadan önce hassas bilgileri kaldırın.
